== Description ==

This plugin is for bookmarking your post into stumble,technorati, digg and delicious. You have to do nothing but just activate the plugin. It automatically add the bookmarking options just below to your posts.

Features:

* Traffic : This plugin increses your website traffic
* Cute Look: Transparent plugin will give your blog a nice look
* Huge Icons:Since the bookmarking is implemented with huge images, the chances for bookmarking are very high

**For Example and Details Visit :** http://www.sajithmr.me/addtothis-wordpress-social-bookmarking-plugin/

**My Technical Blog is :** http://www.sajithmr.me

== Installation ==

1. Unzip the files 
1. Upload addtohtis directory (containing addtothis.php file with image files) to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress


== Frequently Asked Questions ==


= Should i edit any files? =

No, not needed, just activate the plugin.

= Images are not displaying? =

Make sure the path to main plugin file is  /wp-content/plugins/addtothis/addtothis.php.
